Programación Web de un Portafolio Personal de Proyectos

Aplicación de IA para el diseño web

1. Alcance del proyecto.

Este proyecto tuvo como objetivo desarrollar un portafolio web para un gerente de proyectos, permitiéndole presentar sus logros y habilidades. Se adaptó una plantilla descargada de telegram, personalizando los archivos HTML, CSS y JavaScript para reflejar su información profesional y personal. También se creó una hoja de rutas en Python para manejar la navegación entre proyectos, y se diseñó una página individual para cada proyecto, logrando una interfaz organizada y atractiva. Se colocó una nueva ventana (esto fue lo mas difícil) para que el cliente colocara los servicios de su emprendimiento de empresa de ingeniería, además de otros servicios que presta.

Muestra del Proyecto

2. Fases del desarrollo web asistido por IA.

El desarrollo de programación asistido por IA incluyó varias fases: recolección de requerimientos, personalización, desarrollo, y pruebas. En la fase de personalización, se utilizó chatPGT 4o (modelo canvas) y GitHub Copilot, estos fueron clave para adaptar el diseño y los datos. En el desarrollo, se implementaron rutas en Python para la navegación, y en las pruebas, se ajustaron los elementos visuales y de funcionamiento en distintos dispositivos, todo de manera eficiente gracias a la IA.

3. Ventajas del diseño web asistido por IA.

La IA permitió reducir tiempos y mejorar la calidad, asistiendo en la escritura y corrección de código. Herramientas como GitHub Copilot facilitaron la personalización del diseño y optimizaron la resolución de errores en tiempo real, ofreciendo recomendaciones y mejorando la calidad visual y funcional del portafolio.

4. Galería del Proyecto

4. Herramientas y técnicas del PMI para la gestión ágil del proyecto

Se aplicaron técnicas ágiles del PMI como sprints, reuniones diarias y backlog, permitiendo una organización flexible y eficiente. Los sprints semanales se centraron en cada etapa clave (diseño, desarrollo y pruebas), mientras que las reuniones ayudaron a resolver problemas y ajustar prioridades. El uso de un tablero Kanban hizo visible el progreso del trabajo, logrando un flujo ordenado y adaptable a los cambios.
